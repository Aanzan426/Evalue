# Evalue Finance Tracker

Evalue is a Python-based finance tracker that uses JSON for storage.
It helps monitor personal finances, track savings, and analyze spending patterns.

## Current Features
- Track daily expenses and savings
- Store data in JSON format
- Simple CLI-based interaction

## Future Plans
- Integrate stock market APIs to monitor investments
- Provide monthly suggestions and reviews for financial evaluation
- Build a web interface for visualization and dashboards
- Expand to portfolio optimization and investment analysis

## Tech Stack
- Python
- JSON (data storage)
